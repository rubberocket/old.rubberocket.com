rubberocket.com is a website maintained and created by Aleph Diallo.
rubberocket contains some stuff, like games, screensavers, cartoons and such and such. If you don't like it, there's a bunch of other sites you might like, such as YouTube and Twitter.

list of site influences

  • twelvemen - twelvemen.neocities.org
  • radiohead.com (2000 version) - archive.radiohead.com/2000
  • newgrounds - newgrounds.com
  • drevonor.com (specifically when it was called razorback)
  • randall and friends - ranfren.neocities.org
  • every single geociites website ever (no, really!!)

rubberocket is powered using static HTML, meaning that JavaScript is used. If you don't have that enabled, you can't really view this site with the same amount of ease as with a browser with JavaScript enabled.

the webserver being used is netlify. formerly, it was being powered by github pages, but that sucked like dogpiss!

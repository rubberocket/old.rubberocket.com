what's rubberocket*?

glad you asked.

rubberocket* originally started off as RubberRocket Studios, a bad-looking website made under Wix. The site would eventually be abandoned after a while and would eventually become rubberocket*. That's all you need to know.

can I visit this site on an old computer?

Kinda.
You're going to need the following:
For Windows 7+: Pale Moon, Windows 7 SP1 or newer.
For Windows XP/Vista: MyPal Browser, Windows XP SP2 or newer. The 64-bit version is recommended. Or you can use the One Core API with Chrome, that works, too, I suppose.
For Windows 95, NT 4, 98/ME & 2000: Retrozilla, 256 colors or more.
For Windows 3.1: The only place you can visit is games.rubberocket.com. If you really want to use Internet Explorer, use version 3. On a Mac? You should definitely find a Firefox browser compatible with your OS. If you're using MacOS 10.7 or later, see "For Windows 7+".

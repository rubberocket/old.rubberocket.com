can I visit this site on an old computer?

Kinda.
You're going to need the following:
For Windows 7+: Pale Moon, Windows 7 SP1 or newer, and (optionally) VLC.
For Windows XP/Vista: MyPal Browser, running on Windows XP SP2 or newer (64-bit recommended), and (optionally) VLC. Or you can use the One Core API with Chrome, that works, too, I suppose.
For Windows 95, NT 4, 98/ME & 2000: Internet Explorer might work, but I'd recommend you use Retrozilla or Firefox 2. You also need a monitor capable of 256 colors or more and VLC with KernelEx installed.
For Windows 3.1: Use Internet Explorer version 4 or 5.
For MacOS 10.0+: You should definitely find a Firefox browser compatible with your OS. You will also need VLC, download the version compatible with your OS version. If you're using MacOS 10.7 or later, see "For Windows 7+".

Why is this site licensed under the Creative Commons?

It's easier to give people permission using Creative Commons than with copyright. The rules are easy to understand, and I'd easily allow you to do anything that the license allows you to do.
Of course, there are some times where the license changes, especially if it has characters I don't own.

How do you make stuff like this?

I animate in Flash Adobe Animate, but I'm starting to learn OpenToonz.
I code HTML without a WYSIWYG, and I use Git to make all this stuff public.
I also use Linux. I've had some experience with Gentoo and Void, but Debian just has better support for things.
This website was originally hosted on GitHub, but it's now on GitLab for reasons.
